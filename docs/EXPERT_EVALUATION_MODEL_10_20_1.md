## 2. 테스트 시나리오 (3회×3회 = 최소 9회 테스트)

### 시나리오 A: Train 기본 플로우
1. Unmasked/Masked/Output 폴더 선택
2. Apply suggested 클릭
3. Start training (1 epoch)
4. 그래프·로그 갱신 확인

### 시나리오 B: ROI Editor
1. Edit ROI 클릭
2. 이미지 표시, 드래그로 ROI 지정
3. Prev/Next, Save ROI map

### 시나리오 C: Inference
1. 학습된 모델 경로 지정
2. 이미지 선택, Run inference

### 시나리오 D: EXE 실행
1. 일반 권한으로 EXE 실행
2. 위 A/B/C 시나리오 수행
