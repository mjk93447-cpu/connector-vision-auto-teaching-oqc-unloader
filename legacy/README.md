# Legacy

애매한 파일·아티팩트를 보관. 나중에 판단 후 삭제 또는 복구.

- **target_score_strategy_config.json** — `tools.py tune --batch` 실행 시 자동 재생성됨. 참고용.
- **download_exe.bat** — GitHub Artifacts 다운로드용. 필요 시 복구.
- **run_tests_standalone.py** — `run_tests.bat`은 `python -m unittest discover` 사용. 중복.
- **VERSION** — 버전 번호(22). 필요 시 README/VERSION 파일로 복구.
